📱 Panduan Termux

Panduan lengkap penggunaan Termux — terminal emulator & lingkungan Linux untuk Android. Ikuti langkah-langkah berikut untuk setup Termux dari awal.

Install Termux dari F-Droid (bukan Play Store)

1
Install Termux
Download Termux dari F-Droid untuk versi terbaru yang terus diperbarui. Jangan gunakan versi Play Store karena sudah tidak di-maintain.
https://f-droid.org/en/packages/com.termux/

Update paket setelah install pertama kali

2
Update & Upgrade Paket
Setelah install, jalankan perintah update untuk memperbarui semua repository dan paket ke versi terbaru.
pkg update && pkg upgrade -y

Install Python & tools dasar

3
Install Paket Esensial
Install Python, Git, dan paket penting lainnya yang sering dibutuhkan untuk pengembangan.
pkg install python git curl wget -y

Setup storage akses ke file Android

4
Setup Akses Storage
Izinkan Termux mengakses penyimpanan Android agar bisa membaca/menulis file di SD Card dan Internal Storage.
termux-setup-storage

Install Node.js untuk development web

5
Install Node.js
Untuk pengembangan web dan JavaScript, install Node.js beserta npm package manager.
pkg install nodejs-lts -y node --version npm --version

Tips keamanan & penggunaan Termux

6
Tips & Keamanan
Perintah-perintah penting yang perlu diketahui untuk penggunaan Termux sehari-hari secara aman.
# Lihat proses berjalan ps aux # Keluar dari sesi exit # Clear layar clear
